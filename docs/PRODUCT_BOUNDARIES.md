# Product Boundaries

NetSuite Demo Helper should stay focused on practical SC demo preparation and demo intelligence.

## The Product Is

- Demo intelligence
- SC preparation
- Storytelling support
- Rehearsal support
- Discovery gap detection
- Demo risk reduction
- NetSuite demo flow guidance

## The Product Is Not

- A CRM
- A full ERP implementation tool
- A BI platform
- A project management system
- A replacement for SC judgment
- A source of verified competitive claims

## Operating Principle

The application can organize, prioritize, and contextualize demo preparation. It should not pretend to replace discovery, customer validation, approved competitive materials, or the judgment of an experienced Solution Consultant.
